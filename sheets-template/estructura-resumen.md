# Estructura de Google Sheets — Hoja "Resumen"

Crear una hoja llamada `Resumen` con las siguientes columnas en orden:

| Columna | Tipo | Descripción |
|---|---|---|
| FECHA | Texto | Fecha de recepción del email |
| REMITENTE | Texto | Dirección de email del remitente |
| ASUNTO | Texto | Asunto del correo |
| CATEGORÍA | Texto | Consulta / Soporte / Urgente / Requiere revisión |
| PRIORIDAD | Texto | alta / media / baja |
| RESUMEN | Texto | Resumen generado por el LLM (máx. 150 caracteres) |
| DOCUMENTOS ADJUNTOS | Texto | SI / NO |
| RESPONDIDO | Texto | SI / NO |
| FECHA RESPONDIDO | Texto | Fecha en que se respondió el email |
| RESUELTO | Texto | SI / NO |
